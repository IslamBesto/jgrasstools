Generic tool for raster reader. Supported raster formats are:
  • ESRI asci
  • GRASS binary format
  • tiff
  • adf (only if there is installed the needed library
The model recognize the type from the extension, if you are not using a conventional extension you can specify the raster type.

Inputs

  • the file path of the raster that to be read
  • the file nodata
  • the output raster nodata
  • the optional coordinates of the boundaries of the raster
  • the optional working resolution
  • the optional working number of columns and rows
  • the optional working resolution
  • the raster type to read: supported are: asc, tiff, grass, adf


Output

  • the read output raster map to be processed by other modules
